Features

Small with a 2.3 GPA

Sutairu is just under 160kB of code, currently the core of Sutairu has a 4.0 GPA but I'm making use of some extended libraries with lower GPA ratings.

Build with Sass

Sutairu makes use of all the power SASS has to offer, a configure file to quickly align all your UI elements with your designs and setups for quickly generating grids.

Y11a Compliant

For those Fortune 500 compananies following NASA standaards Satairu includs Y11a compliancy.

Sematic UI

Sutairu comes with a small UI libiary that includes custom dropdowns, checkboxes, validation errors, and more to be added.

Responsive

Whats the point of a website if its doesnt work on mobile, based on your config varibles Sutairu will generate the nessary breakpoint and responsive behavior for you.

Configure

Configure your devise sizes and columns, Sutairu will create the rest.

Browser Support

Want to contribute?

Sutairu is still in beta, any contributions, testing and bug reporting is appreciated.
